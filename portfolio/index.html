Projetos

Extrator DAS

Desenvolvi uma solução de automação em Python que transforma o processo manual de extração de dados de PDFs de DAS em uma tarefa de segundos, utilizando Pandas e RegEx para garantir precisão e gerando relatórios em Excel de forma automática.

Ver Demo

Horas Pro

Gerenciador de horas de estágio completo com backend Python/Flask. Salve seus dados de forma segura e privada em um banco de dados local. Inclui dashboard e relatórios. Disponível em duas versões: uma completa com Python/Flask (para uso local) e uma demo online em JavaScript puro.

Ver Site GitHub

Projetos Acadêmico

CESAR COMISC SHIELD

Uma recriação do icônico Space Invaders, programada inteiramente em C. Neste jogo de tiro para terminal, o jogador controla uma nave para derrotar hordas de inimigos que se aproximam, tudo em uma interface de modo texto utilizando a biblioteca Cli-Lib.

Ver Demo GitHub

CondFacil

Sistema de gestão de condomínios que otimiza a administração de moradores, finanças e áreas comuns. O projeto demonstra a implementação de Programação Orientada a Objetos em Java com arquitetura MVC, utilizando PostgreSQL como banco de dados.

Ver Demo GitHub

Turismo Connect

Projeto acadêmico que demonstra a construção de um sistema CRUD completo em Python. A aplicação gerencia o ciclo de vida de registros (Criar, Ler, Atualizar, Deletar) em um arquivo CSV, utilizando a biblioteca Pandas para a manipulação dos dados e os princípios da Programação Orientada a Objetos para a estrutura.

Ver Demo GitHub

Quiz Lógica

Projeto de um jogo de quiz focado em desafios de lógica e resolução de problemas. Desenvolvido com HTML, CSS e JavaScript, o site é totalmente responsivo e oferece feedback instantâneo ao jogador. Uma demonstração prática de manipulação do DOM e desenvolvimento front-end.

Ver Site GitHub
